## carousel
- 100% html, css, javascripts만 사용하였으며 모두 직접 만들었습니다.
- [프로젝트 링크](./inan_carousel)
- 간단한 소개
  - 7개의 원이 다음 아이템 위치로 이동합니다.
  - 처음 위치 아이템은 opacity를 1로 마지막 아이템으로 가는 아이템은 0으로 설정됩니다.
  - 다음 이벤트는 2초마다 무한반복됩니다.
